# Ses dosyaları

Her harf için bir MP3 ekleyin. Dosya adları **küçük Türkçe harf** olmalı:

| Harf | Dosya adı   |
|------|-------------|
| A    | `a.mp3`     |
| Ç    | `ç.mp3`     |
| Ğ    | `ğ.mp3`     |
| İ    | `i.mp3`     |
| I    | `ı.mp3`     |
| Ö    | `ö.mp3`     |
| Ş    | `ş.mp3`     |
| Ü    | `ü.mp3`     |

Örnek: `sounds/a.mp3` — "A" harfinin telaffuzu.

Dosya yoksa oyun geçici olarak tarayıcının metin-konuşma (TTS) özelliğini kullanır.

**Kaynak önerisi:** Kendi kaydınız, [Freesound](https://freesound.org) veya eğitim materyali ses paketleri.
